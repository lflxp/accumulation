## 算法在计算中的作用

算法应用广泛，包括解决一些问题，以及让解决一个问题更快速，更省空间。算法的解决哪些问题：

* 基因工程
> DNA中有十万个基因，30亿个化学基对，数据量很大

* 互联网
> 管理和处理海量数据（路由，搜索引擎）

* 电子商务
> 加密（公钥密码和数字签名）

* 资源分配
> 最大利益（线性规划）

* 交通
> 导航（最短路径）

* 基因匹配
> 最长公共子序列

* 依据部件库的机械设计
> 拓扑排序
